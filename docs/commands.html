üêç Python Commands

Specialized commands for Python development, testing, and optimization.

/test-suite-gen
Generate comprehensive test suites with high coverage
/test-suite-gen component=UserAuth coverage_target=95 include_integration=true

Arguments:

  • component - Component/module to test
  • coverage_target - Target test coverage percentage
  • include_integration - Include integration tests
  • test_framework - pytest, unittest, or nose

Example:

/test-suite-gen component=PaymentProcessor coverage_target=90 test_framework=pytest
python
/performance-heatmap
Identify performance bottlenecks with detailed analysis

/keyword-research
Comprehensive keyword research and analysis using multiple data sources
/keyword-research seed_keywords="digital marketing,seo services" target_location=US search_volume_min=500

Arguments:

  • seed_keywords - Initial keywords to expand from
  • target_location - Geographic target (default: US)
  • search_volume_min - Minimum monthly searches (default: 100)
  • difficulty_max - Maximum keyword difficulty (default: 70)
  • include_questions - Include question-based keywords

Example:

/keyword-research seed_keywords="python development,web scraping" target_location=US search_volume_min=1000
