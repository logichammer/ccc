Ubuntu/Debian Installation

# Update package list
sudo apt update

# Install required packages
sudo apt install python3 python3-pip git

# Verify installations
python3 --version
pip3 --version
git --version

macOS Installation

# Install Homebrew if not already installed
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

# Install required packages
brew install python3 git

# Verify installations
python3 --version
pip3 --version
git --version

üöÄQuick Installation

1
Clone or Access CCC System

Ensure you have access to the CCC system directory:

cd /home/loki/code/ccc
2
Run Automated Installation

Execute the installation script which handles all dependencies and setup:

./install.sh
üí° What the installer does:
  • Checks for required dependencies
  • Installs Python packages (toml)
  • Creates external project directories
  • Installs command-line tools globally
  • Sets up shell aliases
  • Verifies everything works
3
Verify Installation

Test that everything is working correctly:

claude-init --list-types
claude-work --help
./verify.sh
‚úÖ Installation Complete! You're ready to start using the CCC system.

üîßManual Installation

For custom setups or troubleshooting, follow these detailed steps:

1
Install Python Dependencies
pip3 install --user toml
2
Create Project Directories
mkdir -p /home/loki/python-projects
mkdir -p /home/loki/seo-projects
3
Install Command-Line Tools
sudo cp /home/loki/code/ccc/bin/claude-init /usr/local/bin/
sudo cp /home/loki/code/ccc/bin/claude-work /usr/local/bin/
sudo chmod +x /usr/local/bin/claude-*
‚ö†Ô∏è Note: You may need administrator privileges for installing to /usr/local/bin/
4
Set Up Shell Aliases (Optional)

Add helpful aliases to your shell configuration:

# For bash users (~/.bashrc) or zsh users (~/.zshrc)
echo 'alias ccc-new="claude-init"' >> ~/.bashrc
echo 'alias ccc-work="claude-work"' >> ~/.bashrc
source ~/.bashrc

    üö® Troubleshooting

    Permission Denied Error

    # If you get permission errors:
    sudo chmod +x /home/loki/code/ccc/bin/claude-*
    sudo chmod +x /home/loki/code/ccc/install.sh

    Python Module Not Found

    # Install missing Python packages:
    pip3 install --user toml

    # Or use system-wide installation:
    sudo pip3 install toml

    Command Not Found

    If claude-init or claude-work commands are not found:

    # Check if /usr/local/bin is in your PATH:
    echo $PATH

    # Add to PATH if needed (add to ~/.bashrc):
    export PATH="/usr/local/bin:$PATH"

    # Or run directly:
    /home/loki/code/ccc/bin/claude-init --help

    Windows Line Ending Issues

    # Fix line endings on Linux/Mac:
    sed -i 's/\r$//' /home/loki/code/ccc/bin/claude-*
    sed -i 's/\r$//' /home/loki/code/ccc/install.sh
